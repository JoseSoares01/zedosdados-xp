          Lista de Boas Práticas (Fazer e Não Fazer)
        

        

          
          
          
        

      

      

        

          

            

              Fazer
            

            
    
              
  • Use Verde Limão (#96FF96) especificamente para indicadores de sucesso.
    • 
              
  • Alinhe o espaçamento exclusivamente à grade base de 4px.
    • 
              
  • Limite o tamanho do raio das bordas dos componentes a 3px (elementos) e 5px (superfícies).
    • 
              
  • Empregue Tahoma para as hierarquias visuais principais.
    • 
              
  • Mantenha altas taxas de contraste de cor tipográfica (acima de 4,5:1).
    • 
            

          


          

            

              Não Fazer
            

            
    
              
  • Não misture fontes serifadas e não-serifadas dentro da mesma página de layout.
    • 
              
  • Não exceda a largura padrão de botões de desktop (94px) sem contexto direto.
    • 
              
  • Evite múltiplas camadas de sombra sobrepostas.
    • 
              
  • Não use sistemas de redimensionamento de texto fluidos ou relativos.
    • 
              
  • Nunca use cores de destaque como amarelo ou verde para fundos de containers amplos.
    • 
            

          

        

      

    



    
    

      

        

          
          Simulador de Breakpoints Responsivos

          as colunas colapsam em linhas únicas, o espaçamento diminui e os botões ocupam 100% de largura.
